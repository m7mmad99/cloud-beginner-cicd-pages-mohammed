What is CI/CD?

CI/CD stands for Continuous Integration and Continuous Deployment. It is a modern development practice that automates building, testing, and deploying applications. When developers push code to a repository, the system automatically checks the code and deploys updates. This process helps teams deliver software faster and with fewer errors.

My Pipeline

When I push my code to the GitHub repository, GitHub Actions automatically runs the workflow. It checks that all files exist and then deploys the website to GitHub Pages. After the workflow finishes, the updated website becomes live online.
